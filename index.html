HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements nested inside previous elements have a child-parent relationship. Nesting elemnents are the child elements.
  • Questions:
    • Do child elements have a minimum of three?
    • Does a parent element need a minimum of three elements nesting inside?
  • The difference between elements and tags: the element is the entire element from opening to end (including conent?, etc.), the tag is only what appears in the angle brackets
